# HuffmanCompressor (C++)

A simple implementation of Huffman coding for text compression.

This version focuses on demonstrating:
- Frequency counting
- Building a Huffman tree
- Generating prefix codes
- Encoding and decoding strings in memory

## How to Build & Run

```bash
cd src
g++ main.cpp -o huff
./huff
```

You will be prompted to enter a string, which will be encoded and then decoded.
